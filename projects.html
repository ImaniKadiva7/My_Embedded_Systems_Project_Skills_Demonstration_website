Reading Push Button Project

As seen on the project title above

This project reads the state of the pushbutton

When pressed, the button state changes from

High to Low and the LED turns off and vice versa is true

×

int led = 13; // LED connected to pin 13

int buttonPin = 7; // button pin connected to pin 7

int storageValue = 0; // variable to store the read value


void setup() {

Serial.begin(9600); // initialize the serial monitor

pinMode(led, OUTPUT); // set the LED as output

pinMode(buttonPin, INPUT); // set the button as input

}

void loop() {
// put your main code here, to run repeatedly:
storageValue = digitalRead(buttonPin); // set value to the button input
digitalWrite(led, storageValue); // set LED to the pushbutton value
}
